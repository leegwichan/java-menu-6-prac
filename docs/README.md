# 기능 구현 리스트

- 입력
    - 코치 이름
        - 검증
            - 입력 형식에 맞지 않는 지(한글과 `,`만 허용)
            - 이름이 2글자 ~ 4글자인지
            - 코치가 2 ~ 5명인지
    - 못 먹는 메뉴
        - 검증
            - 입력 형식에 맞지 않는 지(한글과 `,`만 허용)
            - 0 ~ 2개인지
    - 잘못된 값 입력시 에러 출력 후 다시 받기
- 출력
    - 추천 시작
    - 메뉴 추천 결과
    - 추천 완료
- 각 코치들이 먹지 못하는 메뉴 저장
- 특정 요일에 추천할 카테고리 선택
    - 카테고리 목록 가져오기
    - 같은 카테고리가 3번 이상 중복되지 않도록
- 해당 카테고리에 맞는 메뉴 추천
    - 특정 카테고리에 해당하는 메뉴들 가져오기
    - 중복되지 않도록
